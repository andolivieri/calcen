# calcen

Solver for game *Calcolo Enigamtico", based on JST permutation generation algorithm

# Usage

java it.andolivieri.calcoloenigmatico.main <input-file>

# Input file

Must be something like:

'''
AB * C = DBC
FGA / D = DD
BGF + FB = BAE
AB + FGA = BGF
C * D = FB
DBC - DD = BAE
AB + C = EHJ
'''
